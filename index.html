Sequential Column Mapping

How does column mapping work?
‚Ä¢ Each spreadsheet column maps to Week 1, Week 2, Week 3, etc. in order
‚Ä¢ No complex date parsing required - works with any column labels
‚Ä¢ Consistent mapping across all calculations and tabs

Workflow Recommendations

1. Upload your spreadsheet in the Spreadsheet Mapping subtab
2. Review the automatic sequential column mapping (Week 1, Week 2, etc.)
3. Optionally set a base year for date calculations
4. Review cashflow in Weekly Cashflow tab to verify data accuracy
5. Check profitability in Profit & Loss tab
6. Use Investment Return & Repayments for ROI analysis and repayment planning
7. Review consolidated results in Summary tab
